Let's see how Doctor Kyek Api works


Learn with Video

Create

To create an article, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles/create) with POST method by passing four data (title,img_link,content,writer_name).

Retrieve

To get all articles, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles) with GET method.


To search a specific article with id, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles/id) with POST method by passing article id.


To search articles with title, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles/title) with POST method by passing title.


To search articles with name of a writer, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles/writer_name) with POST method by passing the writer's name.

Update

To update an article, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles/update) with POST method by passing these data (id,title,img_link,content,writer_name).
No need to pass all of them except id. You only need to pass the data you want to change.

Delete

To delete an article, call this endpoint (http://kaungkhantthukkt2016.000webhostapp.com/articles/delete/id/{id}) with GET method by passing article id.

Data Order in body of POST method doesn't matter.
